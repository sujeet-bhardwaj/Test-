# 🌦️ Weather App

A simple weather application built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data from the **OpenWeatherMap API** and displays temperature, humidity, and city details.

---

## 🚀 Features
- Search weather by city name
- Displays:
  - 🌡️ Temperature (in °C)
  - 💧 Humidity
  - 🏙️ City name
- Error handling for empty or invalid inputs
- Responsive UI with show/hide sections

---

## 🛠️ Tech Stack
- **HTML** → Structure of the app
- **CSS** → Styling and layout
- **JavaScript** → Logic, DOM manipulation, API calls
- **OpenWeatherMap API** → Weather data provider

## ⚡ How It Works
1. User enters a city name in the input box.
2. JavaScript fetches data from OpenWeatherMap API:
3. Weather details are displayed dynamically on the page.
